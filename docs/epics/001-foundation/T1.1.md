# Monorepo scaffold & tooling

description: Initialize the pnpm + Turborepo monorepo with `apps/api` (Express+TS),
`apps/web` (React+Vite+TS), and `packages/contracts`. Configure TypeScript
(strict), ESLint, Prettier, and the lint rule forbidding LLM provider SDK imports
outside `apps/api/src/llm/adapters/*`. Provide root scripts (build/lint/typecheck/test).

tag: infra

acceptance criteria:
- `pnpm install && pnpm build` succeeds across all workspaces.
- `pnpm lint` and `pnpm typecheck` pass; `strict: true`, no `any`.
- The custom lint rule fails a build if a provider SDK is imported outside the
  adapters folder.
- Folder layout matches conventions/README.md.

  conventions: [naming, testing]
  tech-stack:  [typescript, express, react]
