# Shared contracts package (Zod schemas + types)

description: Implement `packages/contracts` as the single source of truth for the
FE/BE seam: Zod schemas + inferred TS types for the chat/SSE events, ResultEnvelope,
PermissionBlock, RBAC model (Role, ResourceGrant, User, DataSource), GeneratedQueryView,
AuditEvent, auth payloads, and the `error-codes` discriminated union. Export typed
helpers to parse/validate at the edge.

tag: shared

acceptance criteria:
- Every shape in contracts.md is exported as a Zod schema + inferred type.
- Both `apps/api` and `apps/web` import from `@bi/contracts` with no duplication.
- Unit tests assert representative valid/invalid payloads parse/reject correctly.
- The `error-codes` union and event-shape unions are exhaustive (compile-time check).
