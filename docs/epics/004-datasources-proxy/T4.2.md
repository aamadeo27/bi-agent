# SQL connectors (PostgreSQL, MySQL, BigQuery)

description: Implement a `Connector` interface and SQL adapters using `pg`,
`mysql2`, and `@google-cloud/bigquery`. Each adapter: connect with a supplied
credential, run a validated read-only query with a statement timeout + row cap,
introspect schema (schema>table>column+types for the schema-tree endpoint), and
normalize rows/columns into the ResultEnvelope shape (column type + role inference).
Include a `testConnection()`.

tag: backend
deps: T4.1

acceptance criteria:
- All three adapters implement `Connector` (connect, query, introspect, test).
- Queries run with timeout + row cap; results normalized to envelope columns/rows.
- Schema introspection feeds T3.2's schema-tree endpoint.
- Tests use Testcontainers (pg/mysql) and a mocked/recorded BigQuery client.
