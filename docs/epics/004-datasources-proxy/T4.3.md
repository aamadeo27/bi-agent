# Generic REST connector (token auth + declared schema)

description: Implement the generic REST `Connector` (using `undici`): a connection
declares base URL, auth token, and a **declared endpoint/field allow-list** that
serves as both the "schema" (for grants + the schema-tree endpoint) and the
validation surface. Execute a read request against an allow-listed endpoint, apply
timeout + result size cap, and normalize the JSON response into the ResultEnvelope
row/column shape. Include `testConnection()`.

tag: backend
deps: T4.1

acceptance criteria:
- REST connector implements `Connector`; declared endpoints/fields drive grants.
- Only allow-listed endpoints/fields are reachable; others rejected.
- JSON responses normalize into envelope columns/rows with inferred types.
- Tests: allow-list enforcement, normalization, timeout (mock HTTP server).
