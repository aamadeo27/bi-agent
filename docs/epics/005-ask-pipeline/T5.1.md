# LLM provider abstraction + Gemini adapter

description: Define the `LlmProvider` port (`streamText`, `generateQuery`, `id`,
`model`) and a `createLlmProvider(config)` factory selecting an adapter from
`LLM_PROVIDER`/`LLM_MODEL`. Implement the default `GeminiProvider` using
`@google/genai` (the only file allowed to import it). `generateQuery` returns a
deterministically-parseable `QueryProposal`. Provide a `MockLlmProvider` for tests.

tag: backend
deps: T1.3

acceptance criteria:
- Port + factory + Gemini adapter + mock implemented per patterns provider-abstraction.
- Provider switch is config-only; no pipeline code references a concrete SDK.
- Lint rule confirms no `@google/genai` import outside the adapter.
- A recorded-fixture contract test validates the Gemini adapter's output shape;
  pipeline tests use the mock.
