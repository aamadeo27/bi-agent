# Chart components (bar/line/pie/table) + a11y

description: Implement the four Recharts-based chart components (bar, line,
pie/donut) and the data table (sortable headers, pagination 20/50/100, alternating
rows) driven by the ResultEnvelope. Apply the categorical palette, hover tooltips
(FR-VIZ-5), and the accessibility per UI/UX §10: chart SVG `role="img"` + summary
`aria-label`, per-slice/element labels, keyboard-focusable elements, table as the
accessible fallback with proper `<th scope>`/`<caption>`.

tag: frontend
deps: T1.4

acceptance criteria:
- All four chart types + table render from a ResultEnvelope with correct palette.
- Tooltips on hover and on keyboard focus where supported; values formatted.
- A11y per §10 (role/img, aria-labels, accessible table); axe passes.
- Empty-result and large-result (sample/aggregate) treatments per §11.
- RTL tests per chart type.
