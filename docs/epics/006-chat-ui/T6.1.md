# Chat workspace shell (S2): layout, sidebar, history

description: Build the S2 workspace shell: three-column layout, left conversation
sidebar (new-conversation, list with titles/relative timestamps, active highlight,
hover delete), top nav bar (conversation title editable, admin link for admins, user
menu), and the sticky input bar (auto-expanding textarea, Enter/Shift+Enter,
send-enabled-on-nonempty, disabled-while-streaming). Wire to the conversations API.

tag: frontend
deps: T1.4, T2.6

acceptance criteria:
- S2 layout + sidebar + top nav + input bar match UI/UX §6/§7/§8.
- New/select/delete conversation works against the API; titles derive from first
  message; empty/no-history states render.
- Keyboard behavior + WCAG 2.1 AA focus/contrast; RTL + axe tests pass.
